
> **Note:**  
> Each folder focuses on a specific JavaScript topic, with code files and sometimes HTML for hands-on practice.

---

## 📝 How to Use

1. **Clone the repository:**
    ```bash
    git clone https://github.com/FaizanInstinct/javascript-complete.git
    cd javascript-complete
    ```

2. **Browse the folders:**  
   Each folder contains code samples and exercises for a particular topic.

3. **Open `.js` files in your favorite editor:**  
   You can run them in your browser console or with [Node.js](https://nodejs.org/).

4. **Try the HTML demos:**  
   Some folders (like `Farhenheit to Celcius/`) include `index.html` files for interactive practice.

---

## 📚 Topics Covered

- **JavaScript Basics:** Variables, Data Types, Operators, Expressions
- **Control Flow:** If/Else, Switch, Loops (for, while, do-while)
- **Functions:** Declarations, Expressions, Arrow Functions, Scope, `this`
- **Arrays & Objects:** Creation, Methods, Iteration, JSON
- **DOM Manipulation:** (in relevant folders)
- **Mini Projects:** Practical exercises like temperature converters, calculators, etc.
- **Advanced Concepts:** (as you progress)
